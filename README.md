# Minestom Base
A base for Minestom servers, providing useful utilities and features to speed up development.
## Setup project
### Windows
```shell
mkdir run/
./gradlew.bat build
```

### Linux/macOS
```shell
make setup
```
## Reinit project
### Windows
```shell
rm -rf run/
mkdir run/
./gradlew.bat build
```
### Linux/macOS
```shell
make reinit
```
